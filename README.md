# 企业门户 - React 版本

这是一个基于 React、TypeScript 和 Tailwind CSS 构建的现代化企业门户网站演示。

## 技术栈

- ⚛️ **React 18** - UI 框架
- 📘 **TypeScript** - 类型安全
- 🎨 **Tailwind CSS 4.0** - 样式框架
- 🎯 **lucide-react** - 图标库
- 🎭 **shadcn/ui** - UI 组件库

## 功能特性

- ✅ 响应式设计（支持桌面和移动端）
- ✅ 现代化的 UI/UX 设计
- ✅ 导航栏（含移动菜单）
- ✅ Hero 横幅区域
- ✅ 功能特性展示（6个模块）
- ✅ 数据统计面板
- ✅ 新闻资讯卡片
- ✅ 完整的页脚信息
- ✅ 流畅的动画和过渡效果

## 项目结构

```
.
├── App.tsx              # 主应用组件
├── components/          # 所有 React 组件
│   ├── Header.tsx       # 导航栏
│   ├── Hero.tsx         # 首页横幅
│   ├── Features.tsx     # 功能特性
│   ├── Statistics.tsx   # 数据统计
│   ├── News.tsx         # 新闻资讯
│   └── Footer.tsx       # 页脚
└── styles/
    └── globals.css      # 全局样式
```

## 在 Figma Make 中使用

这个项目是在 Figma Make 中创建的，支持实时预览和编辑。直接在 Figma Make 中打开即可查看效果。

## 组件说明

### Header
- 响应式导航栏
- 移动端汉堡菜单
- 搜索、登录、注册按钮

### Hero
- 主视觉横幅
- CTA 按钮组
- 数据统计概览
- 特色图片展示

### Features
- 6个核心功能卡片
- 图标 + 标题 + 描述
- 悬停动画效果

### Statistics
- 4个关键数据指标
- 渐变背景设计
- 趋势标识

### News
- 3篇新闻文章卡片
- 图片 + 摘要 + 日期
- 分类标签
- 悬停缩放效果

### Footer
- 公司信息和联系方式
- 多列链接导航
- 社交媒体图标
